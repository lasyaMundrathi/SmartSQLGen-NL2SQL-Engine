### File Structure:

```bash
project/
│
├── main.py
├── database.py
├── genai.py
├── utils.py
├── .env
└── requirements.txt
```


### Running the App:
1. Install the dependencies using pip install -r requirements.txt.
2. Ensure the .env file is present and contains your Google API key.
3. Run the Streamlit app with the command: streamlit run main.py.
